# RSR

RSR is Reserve Rights, the Reserve ecosystem governance token.

## Current Addresses

| Chain            | Chain ID | RSR Address                                  |
| ---------------- | -------- | -------------------------------------------- |
| Ethereum mainnet | `1`      | `0x320623b8e4ff03373931769a31fc52a4e78b5d70` |
| BSC              | `56`     | `0x23f72a3db61d6cb8abe5d9af1ac4b6c99327bfee` |
| Base             | `8453`   | `0xab36452dbac151be02b16ca17d8919826072f64a` |

Do not use the deprecated old BSC RSR address `0x8762db106b2c2a0bccb3a80d1ed41273552616e8`.

These addresses are documented for integration context. The SDK does not currently export RSR address constants; use `getAddress` from viem if your app needs checksum casing.

## Index DTF Gotcha

Index DTF vote-lock vaults can use different underlying tokens. Do not assume every Index DTF vote-lock token uses RSR.
